# WAVE Player — PWA Аудиоплеер

## Поддерживаемые форматы
MP3, WAV, OGG, FLAC, AAC, M4A, OPUS, WEBM, MP4, WMA, AIFF

## Функции
- 🎵 Воспроизведение любых аудиофайлов
- 📋 Создание плейлистов
- 🔀 Случайный порядок (shuffle)
- 🔁 Повтор (всё / один трек)
- 🔊 Регулировка громкости
- ⏱ Перемотка
- 📱 Мини-плеер при переключении вкладок
- 💾 Работает офлайн

---

## Установка на телефон (Android)

### Способ 1 — Через браузер (рекомендуется)
1. Залейте файлы на хостинг (GitHub Pages, Netlify, Vercel — бесплатно)
2. Откройте сайт в Chrome на телефоне
3. Нажмите ⋮ → «Добавить на главный экран»
4. Приложение установится как обычное!

### Способ 2 — Локально через USB
1. Установите [Android Studio](https://developer.android.com/studio) или простой HTTP-сервер
2. Запустите: `npx serve .` в папке с файлами
3. Откройте `http://ВАШ_IP:3000` в Chrome на телефоне
4. Добавьте на главный экран

### Способ 3 — Через GitHub Pages (бесплатно, 5 минут)
1. Создайте аккаунт на github.com
2. Новый репозиторий → загрузите эти 3 файла
3. Settings → Pages → Deploy from branch → main
4. Ваш плеер доступен по адресу: `https://USERNAME.github.io/REPO/`

## Установка на iPhone (iOS)
1. Откройте сайт в Safari
2. Нажмите кнопку «Поделиться» (квадрат со стрелкой)
3. «На экран Домой»
4. Готово!
